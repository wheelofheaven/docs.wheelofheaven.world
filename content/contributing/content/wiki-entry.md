## Frontmatter

```toml
+++
title = "Term Name"
description = "Definition and context (150-160 chars)"
template = "wiki-page.html"

[extra]
claim_type = "direct"    # direct | inferred | speculative
alternative_names = ["Alias 1", "Alias 2"]
category = "Category"
+++
```

## Structure

1. Opening definition (1–2 sentences)
2. Etymology / origin
3. Main explanation
4. In Wheel of Heaven / Raëlian context
5. See also
6. References (six-source minimum)
