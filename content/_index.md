This site is the **author, contributor, and developer documentation** for the
[Wheel of Heaven](https://www.wheelofheaven.world) knowledge base.

It's a work in progress — Phase 1 of the rollout planned in
[`.claude/plans/docs-site.md`](https://github.com/wheelofheaven/docs.wheelofheaven.world).
The site is live so the deploy chain (Zola → gh-pages → Cloudflare Pages) is
proven before the content migration begins.

## What's coming

- **Getting started** — clone, build, ship in under five minutes
- **Contributing content** — wiki entries, Articles, Newsroom Dispatches,
  library books, translations
- **Contributing code** — Bifrost theme, pipelines, CI, deploy
- **Architecture** — how the ~19-repo org fits together
- **Reference** — frontmatter, file formats, glossary, changelog

Until then, the source material lives in
[`github.com/wheelofheaven/docs`](https://github.com/wheelofheaven/docs).
