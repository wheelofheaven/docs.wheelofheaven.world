## File names

- Markdown: `kebab-case.md`
- SCSS: `_kebab-case.scss` (partials)
- Tera templates: `kebab-case.html`

## Branches

- `main` — production-ready
- `feature/{short-name}` — new work
- `fix/{short-name}` — bug fixes

## Commits

Imperative mood, ≤ 72 chars in the first line. Optional conventional
prefix (`feat:`, `fix:`, `docs:`, `refactor:`, `chore:`).
