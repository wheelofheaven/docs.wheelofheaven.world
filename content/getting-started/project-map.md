## The sites

Three production sites, each its own Cloudflare Pages project:

- `www.wheelofheaven.world` — main knowledge base
- `api.wheelofheaven.world` — JSON API
- `assets.wheelofheaven.world` — image CDN

## The repos

- `bifrost` — Zola theme used by the main site
- `data-content` — markdown content (submodule of www)
- `data-library` — book JSON (submodule of www)
- `data-images` — image processing pipeline
